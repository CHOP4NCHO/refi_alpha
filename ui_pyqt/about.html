REFI ALPHA

Evaluación automatizada de requisitos de software mediante Inteligencia Artificial.

REFI ALPHA es una plataforma diseñada para evaluar la fidelidad entre los requisitos documentados y su implementación en código fuente. Utiliza modelos de lenguaje y agentes especializados para automatizar el análisis de software.

Características principales

  • Extracción de requisitos desde documentos PDF.
  • Análisis y mapeo de repositorios de código fuente.
  • Evaluación automatizada mediante inteligencia artificial.
  • Soporte para múltiples proveedores de modelos de lenguaje.
  • Modos de evaluación Pipeline Directo y Agente Completo.
